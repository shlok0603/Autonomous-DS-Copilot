# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/shlok0603/Autonomous-DataScience-CoPilot.git

cd Autonomous-DataScience-CoPilot
```

---

## Create Virtual Environment

Windows

```bash
python -m venv .venv

.venv\Scripts\activate
```

Linux / Mac

```bash
python3 -m venv .venv

source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Gemini API

Create a `.env` file.

```
GEMINI_API_KEY=YOUR_API_KEY
```

---

## Run Application

```bash
streamlit run frontend/streamlit_app.py
```
